## Project Description

Tereth is described as an "ever-rising asset for your portfolio" - a community token launched on Uniswap in July 2021. Features 10% redistribution to holders on both buys and sells, with 16% total tax (6% to liquidity). Claims include liquidity locked and dev keys burned. The project is positioned as a "relaunch" and connected to the SQWL educational ecosystem.
