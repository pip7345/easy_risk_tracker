## Project Description

ZPLN is a mobile gaming platform that merges entertainment, competition, and crypto rewards. The flagship game is a "psychedelic, space-based target-shooting strategy game" powered by the ZPLN token, described as a "zero-point light node" designed to grow in value as the community expands. Players can mine ZPLN tokens through gameplay, compete for crypto prizes (BONK, LEARN tokens), and participate in blockchain-based events. The platform features free-to-play games with referral incentives and an upcoming paid "Echo" product ($35) with MLM-style compensation.

**Tags:** Play to Earn, Game
