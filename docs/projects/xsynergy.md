## Project Description

Xsynergy describes itself as a "trustless reward engine that turns USDT into daily earnings." The platform offers two mechanisms: Time-Mining ("lock USDT, earn $XSYN proportional to lockup") and Vault Staking ("stake $XSYN, earn USDT from protocol activity"). Claims to be fee-light, fully on-chain, and non-custodial.
